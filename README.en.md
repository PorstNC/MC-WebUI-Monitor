## MC-WebUI-Monitor
Usage InstructionsThis project is developed using JavaScript and HTML, providing web-based startup and monitoring functions for Minecraft.

## Installation Steps
**1. Clone the repository**
   ```bash
   git clone https://github.com/PorstNC/MC-WebUI-Monitor.git
   ```
Check Node.js Version
Node.js 24 is recommended. Node.js 22 or above should work in theory (the minimum supported version is not fully tested).

2.**Install Dependencies**

Double-click install.bat to automatically install the required dependencies.

3.**Prepare Game Resources**
Use your Minecraft launcher to download the version (or any version with a MOD manager), e.g. 1.21.8.
Copy the assets、libraries folders, and the 1.21.8 game folder from your .minecraft directory into the MC-WebUI-Monitor project directory.
Note:If you are using a modded version, the folder is usually named 1.21.8-XXXXX. Please rename it to 1.21.8!
## Startup Instructions
Start WebUI
Double-click start-webui.bat.
Access the Web Page
Open your browser and go to:http://localhost:5173
Start the Game
Use the buttons on the web page and follow the information prompts at the bottom.
